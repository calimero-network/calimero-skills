All Skills

Complete reference for every skill file in this repository

Available Skills

Each skill is a plain text file. Running npx calimero-skills <skill-name> appends it to the correct file for your editor:

calimero-core
Core Calimero concepts: contexts, applications, identities, RPC calls, and the node architecture. Start here for any Calimero work.
rust-sdk
The calimero-sdk Rust macros (#[app], #[logic]), state management, and how to structure WASM application logic.
mero-js
The @calimero-network/mero-js TypeScript SDK — auth, admin API, RPC execution, SSE/WebSocket subscriptions.
mero-react
The @calimero-network/mero-react hook library — MeroProvider, useCalimero, token persistence, SSO hash parsing.
calimero-client-js
The legacy calimero-client JS SDK — RpcClient and SubscriptionsClient pattern used by curb and older apps.
desktop-sso
Tauri desktop app SSO flow — how openAppFrontend() passes JWT tokens via URL hash so apps skip manual auth steps.
admin-dashboard
Admin dashboard architecture — React pages, context management UI, application installation flow, and the runtime node URL pattern.
abi-codegen
WASM ABI codegen with mero-devtools-js — the generate CLI command, ABI JSON format, and generated TypeScript output.

Install All Skills at Once

# Install every skill for Claude Code + Cursor + Copilot
npx calimero-skills all

# Install specific skills
npx calimero-skills calimero-core mero-js rust-sdk

Where Skills Are Written

Claude Code
Appended to CLAUDE.md in your project root (or created if absent).
Cursor
Appended to .cursorrules in your project root.
GitHub Copilot
Appended to .github/copilot-instructions.md.

Run from your project root. Skills are additive — running the same skill twice does not duplicate content.
