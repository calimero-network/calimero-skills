How It Works

Skills inject structured API knowledge into your AI assistant's context files

The Problem

AI coding assistants are trained on public data, but Calimero's APIs evolve rapidly. Without accurate context, assistants will:

  • Invent API methods that don't exist (hallucination)
  • Use deprecated parameter names from older SDK versions
  • Miss Calimero-specific patterns like WASM execution models or URL hash SSO
  • Suggest generic solutions that don't account for node-specific constraints

Skills fix this by injecting the current, accurate API reference directly into the files that AI assistants read.

Skill Files

Each skill is a markdown file in skills/. When installed, its content is appended to the appropriate AI context file:

skills/calimero-rust-sdk/
Rust WASM SDK: macros (#[calimero_sdk::app]), state types, execution model, RPC handlers.
skills/calimero-client-js/
JS/TS client: RpcClient, SubscriptionsClient, auth flow, SSO token reading.
skills/calimero-registry/
App registry: signing keys, manifest format, publish workflow, versioning.
skills/calimero-desktop/
Desktop integration: SSO URL hash tokens, parseAuthCallback(), window management.
skills/calimero-node/
Node operations: meroctl CLI commands, config files, health checks.
skills/calimero-merobox/
Local dev: merobox config, YAML workflows, local NEAR sandbox, CI setup.
skills/calimero-client-py/
Python SDK: create_connection(), create_client(), AuthMode, async patterns.
skills/calimero-abi-codegen/
ABI codegen: manifest format, npx calimero-abi-codegen, programmatic API.

Target Files per Editor

Claude / Claude Code

Writes to CLAUDE.md in your project root. Claude Code reads this file automatically on every conversation start.

npx @calimero-network/agent-skills <skill>

Cursor

Writes to .cursorrules. Cursor includes this in every AI request as system-level instructions.

npx @calimero-network/agent-skills <skill> --cursor

GitHub Copilot

Writes to .github/copilot-instructions.md. Copilot reads this for repository-specific guidance.

npx @calimero-network/agent-skills <skill> --copilot

Installing Multiple Skills

Run the command multiple times to stack skills — each appends to the same file:

# Install Rust SDK + client JS skills for a full-stack project
npx @calimero-network/agent-skills calimero-rust-sdk
npx @calimero-network/agent-skills calimero-client-js
npx @calimero-network/agent-skills calimero-desktop

# Or for the Cursor editor:
npx @calimero-network/agent-skills calimero-rust-sdk --cursor
npx @calimero-network/agent-skills calimero-client-js --cursor

Each skill is self-contained — order doesn't matter. Install only the skills relevant to your project to keep the context file focused.
