Node Skill

Context for operating a Calimero node with merod and scripting it with meroctl

What This Skill Covers

The calimero-node skill gives your AI assistant the full meroctl command reference and context lifecycle knowledge so it can write correct node setup scripts and operator runbooks.

Covered Topics

  • merod init and startup
  • meroctl node / app / context / identity / call
  • Context lifecycle (install → create → invite → join)
  • App installation from registry or local WASM
  • Multi-node setup with namespace/group membership

Install

npx @calimero-network/agent-skills calimero-node

# Recommended for operators:
npx @calimero-network/agent-skills calimero-node calimero-merobox

Node Setup

# Initialise node (creates key material + config)
merod --node node1 init --server-port 2428 --swarm-port 2528

# Start the node
merod --node node1 run
# Node listens on http://localhost:2428

# Register with meroctl (one-time)
meroctl node add node1 /path/to/calimero/home
meroctl node use node1

Context Lifecycle

1. Install app
meroctl app install --path app.wasm — registers the WASM with the node. Returns an application_id.
2. Create context
meroctl context create --application-id <id> — creates an isolated instance. Installing an app and creating a context are two separate steps.
3. Add identity
meroctl identity new --seed <phrase> — creates a keypair. Identities are per-context members.
4. Invite & join
meroctl context invite generates an invite token; the joining node calls meroctl context join --token <token>.
5. Call app
meroctl call --context <id> --method get_posts --args '{}' — executes a WASM method on the context.

Key Rules

App ≠ Context
Installing an application registers the WASM code. Creating a context instantiates it with its own isolated storage and members.
meroctl node use
Set a default node so you don't need --node on every command. Use meroctl node ls to see configured nodes.
--home flag
Use merod --home ./data --node node1 init to specify a custom data directory instead of the system default.
